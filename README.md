# Vibedeck

> Digital index cards. Chunked knowledge, beautifully delivered.

Vibedeck is an open source platform for creating and browsing beautiful, mobile-first card decks authored in plain markdown. The mental model is the physical index card — one focused idea per card, sequenced to build understanding without overwhelm.

It is not a presentation tool. It is a reading and learning experience that happens to work beautifully in a presentation context.
